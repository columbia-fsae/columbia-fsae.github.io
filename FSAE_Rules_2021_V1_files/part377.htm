EV.9.1 Activation‌

EV.9.1.1 The Tractive System must not be able to energize until the Low Voltage (GLV) System is powered up using a specified procedure

EV.9.1.2 The driver must be able to activate the Tractive System from inside the cockpit without external assistance.
