D.12.13 Endurance Scoring

D.12.13.1 Scoring Term Definitions:

 Endurance Run Time - Total Time for both Drivers, minus the Driver Change lap, minus any Mechanical Black Flag Time, plus any Penalty time D.14.2

 Corrected Time = Endurance Run Time + ( DOO * 2 ) + ( OC * 20 )

 Tyour - the Corrected Time for the team

 Tmin - the lowest Corrected Time recorded for any team

 Tmax - 145% of Tmin

D.12.13.2 The vehicle must complete the Endurance Event to receive a score based on their Corrected Time

a. When Tyour < Tmax. the team score is determined as:


Endurance Score =

250 x

( Tmax / Tyour ) -1

+ 25



( Tmax / Tmin ) -1


b. When Tyour > Tmax , Endurance Score = 25

D.12.13.3 When the vehicle does not complete Endurance (DNF), one point* is awarded per lap: Endurance Score = Number of Completed Laps*

* Points per completed lap may be altered based on course length
